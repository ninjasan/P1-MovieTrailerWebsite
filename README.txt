Quick Start
    - Open fresh_tomatoes.html in the browser of your choosing. 

From scratch
    - Install Python
    - Download/Copy project files to a single directory on your machine
    - Run entertainment_center.py from that directory
        - It will automatically open fresh_tomatoes.html in your default browser

What's included
    - entertainment_center.py
    - fresh_tomatoes.py
    - media.py
    - fresh_tomatoes.html
    - this Readme.txt
